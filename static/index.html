Which Colleges Are Worth the Price Tag?

About this Project: This is a visualization of 20 year net return on investment data (ROI) by college

Instructions:

Hover over each bar to see exact ROI and cost data for that college. Filter by one of the following majors for a closer look at the data: Art, Humanities, Computer Science, Mathematics, Business, Economics, Nursing, Life Sciences, Political Science, History, English, Education, or default back to All

See how majors affect ROI

Data was sourced from the 2014 PayScale College ROI Report. For details on how the numbers were sourced and calculated, please see PayScale's methodology page.
